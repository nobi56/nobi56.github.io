AWKのリソース

2019/12/25

仕様・ソース

  • POSIX specification
  • one true awk (NAWK?)
  • MAWK
  • MAWK(2beta?)
  • GAWK

実装やそのバージョン、ビルドオプションにより、仕様に準拠する度合や独自拡張の広がりに違い。例えば、組み込み関数のlengthについて、文字列においてのみ使える実装、あるバージョンから配列にも使えるよう拡張された実装とがある等。

ドキュメント

  • mawk(1)
  • Gawk: Effective AWK Programming
  • The AWK Programming Language, (HN)

MAWKのmanページはドキュメントを含む。13項立ての解説に5つの用例、仕様と実装の互換性に関する記述も付録。

GAWKのドキュメントを読めば大抵の問題は解決(するはず)。実例も豊富で、独自拡張であるところはしっかり区別された記述、マニュアルでありチュートリアル。

The AWK Programming Languageはとにかく楽しい、何よりも楽しいドキュメント。
